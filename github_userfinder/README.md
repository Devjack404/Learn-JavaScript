# GitHub User Finder

A simple web application to search GitHub users using the GitHub API.  
Users can search by username and view basic profile information.

---

## Overview

This project is built as a learning exercise to understand how to:
- Work with APIs
- Handle user input
- Display dynamic data on a web interface

---

## Features

- Search GitHub users by username  
- Search using GitHub ID  
- Fetch and display data from the GitHub API  
- Clean and modern user interface  
- Responsive layout  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- GitHub REST API  

---

## Notes

This project is intended for educational purposes only and is not designed for production use.

Some features depend on third-party APIs (GitHub API). As a result, certain functionalities may not work as expected due to external limitations such as rate limits or API availability.
